Project Info 

Currency Exchange Platform is a real-time web application built with PHP and Laravel, designed to provide secure and efficient currency conversion using live exchange rate APIs. The system supports MMK, USD, Thai Baht, and Japanese Yen, and includes a peer-to-peer (P2P) trading feature that allows users to exchange currencies directly with each other. The platform focuses on accuracy, performance, and a clean user experience, simulating real-world financial exchange workflows.
